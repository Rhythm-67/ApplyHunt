# Lemma Notes

## Core Concepts

### Pod
A self-contained workspace containing:
- Tables
- Agents
- Workflows
- Files
- Permissions
- Apps

### Tables
Structured data stored in the pod.

### Agents
AI workers with defined responsibilities.

### Workflows
Multi-step automations that connect agents, data, and approvals.

### Approvals
Human review points before important actions occur.
